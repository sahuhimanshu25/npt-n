Complete Study Notes: Tutorial on Gephi for Network Visualization
This tutorial teaches how to visualize and analyze social media graphs (e.g., Twitter follower networks) using Gephi, an open-source tool. It covers data import, basic metrics, layout, styling, filtering, and export.
Getting Started with Gephi
Gephi works with graph data containing nodes (users, pages) and edges (relationships like follows or likes). Supported formats include .gml, .graphml, .csv, and Excel.
Import Process:

Go to File → Open and select your file (e.g., the .gml file generated by twecoll).
Gephi shows a summary: number of nodes, edges, and whether the graph is directed.
Data appears in two tables:
Nodes table: ID and attributes (e.g., followers, username).
Edges table: Source, Target, and optional attributes (e.g., weight).


You can edit values directly or import additional spreadsheets to merge data.
Overview Tab: Analysis and Layout
This is the main workspace for exploring the graph.
Compute Statistics (right panel):

Average Degree: How connected nodes are on average.
Network Diameter: Average shortest path between nodes.
Graph Density: How close the graph is to fully connected (higher = denser).
Modularity: Detects communities (groups of tightly connected nodes).
PageRank: Measures node influence.
Connected Components: Finds isolated or linked subgroups.

These metrics generate new attributes (e.g., Modularity Class) that you can use for visualization.
Layout Algorithms:

Graphs often look messy when first loaded. Use force-directed layouts like Fruchterman-Reingold or ForceAtlas2.
Nodes repel each other; connected nodes attract. Run the algorithm until the graph stabilizes, then stop it manually.

Appearance and Customization

Node Size: Map to attributes like Degree or Followers (bigger = more important).
Node Color: Map to Modularity Class (different colors for communities) or other attributes.
Labels: Enable with the "T" button. Customize font, size, and color.
Edges: Change thickness, color, or curvature for better readability.

Interactive Tools (left panel):

Direct Selection: Click a node to highlight it and its neighbors.
Rectangle Selection: Drag to select a group of nodes.
Drag Tool: Manually move overlapping nodes.
Painter: Quickly recolor selected nodes.
Node/Edge Pencil: Add new nodes or edges manually.

Filters and Workspaces

Filters (right panel): Isolate parts of the graph.
Range: Filter by numerical values (e.g., followers between 100–1000).
Partition: Filter by categories (e.g., Modularity Class).
Topology: Filter by structure (e.g., connected components).

Apply filters, then export the sub-graph to a new workspace for focused analysis without losing the original.

Preview Tab: Final Rendering

Fine-tune aesthetics: edge curvature, fonts, background.
Export high-quality images (PNG, SVG, PDF) for reports or presentations.

Data Laboratory Tab: Spreadsheet-like view for raw editing and verification. You can jump between Overview and Data Laboratory to cross-reference nodes.
Practical Tips from the Tutorial

For dense graphs, run layouts and use filters to reduce clutter.
Compute statistics first — they create useful attributes for sizing/coloring.
Export filtered sub-graphs to new workspaces for deeper dives.
Always check node/edge tables to verify data integrity.

Gephi turns raw social media data (from APIs or tools like twecoll) into visual, interactive graphs that reveal communities, influencers, and patterns.
